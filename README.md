A browser version of the game Rock Paper and Scissors

n/a

---

## 🧠 Features

- ➕ **add Books** - add books using a form
- ✅ **creates a new Object** - Used classes to create new objects
- ❌ **remove Books** - option to remove books
- 🎮 **Read/Not Read** - Option to have read or not read (will add)

---

## 🧱 Built With

- **HTML5** — Structure of the page  
- **CSS3** - Styling 
- **JavaScript (ES6)** - Event Listner, Classes, Objects

---

## ⚙️ How It Works

1. Click on the add to library button **Pop up of a form will show up**
2. add all the **details** of the book name, author, pages, read 
3. The newly created object is created and shown on screen
4. Can remove by clicking on the button **will remove in the ui and array**

All interactions are handled dynamically using JavaScript (no page reloads).
